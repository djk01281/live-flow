### ⚙️ 실행 방법

**서버 실행방법**
```bash
# backend 폴더로 이동
cd backend

# dependency 설치
yarn

# 서버 실행 (dev mode, 3000번 포트)
yarn start:dev
```
**클라이언드 실행방법**
```bash
# frontend 폴더로 이동
cd frontend

# dependency 설치
yarn

# 서버 실행 (dev mode, 5173번 포트)
yarn dev
```
